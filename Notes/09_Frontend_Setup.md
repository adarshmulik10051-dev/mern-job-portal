# 🎤 Interview Questions

### 1. Why did you use Vite?

Vite provides faster development and build performance compared to older tools like Create React App.

---

### 2. Why did you use Tailwind CSS?

Tailwind CSS helps build responsive and modern UIs quickly using utility classes.

---

### 3. Why did you use shadcn/ui?

It provides reusable and customizable UI components, reducing development time.

---

### 4. Why did you use React Router?

To navigate between pages without reloading the application.

---

### 5. Why did you use Axios?

To send HTTP requests between the frontend and backend.

---

### 6. Why did you use Redux Toolkit?

To manage application state in a centralized and efficient way.
